# SCM_Project

A simple Java utility tool for SWE 442 (Software Configuration Management).
It demonstrates version control, feature branching, merging (with conflict resolution),
and release tagging using Git/GitHub.

## Features (v1.0)
- Calculator: add/subtract/multiply/divide (safe divide).
- Unit Converter: °C↔°F, km↔miles.
- Text Utilities: reverse text, palindrome check.

## How to Run
```bash
# compile
javac -d out src/com/scmtool/*.java
# run
java -cp out com.scmtool.App
